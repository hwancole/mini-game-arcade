# 🎮 미니 게임 아케이드

순수 HTML/CSS/JavaScript로 만든 **27가지** 클래식 웹게임 모음입니다. 서버·빌드·의존성 없이 브라우저만 있으면 바로 플레이할 수 있어, **GitHub Pages에 그대로 퍼블리시**할 수 있습니다.

- 🌐 **언어 설정** — 메인 페이지 우측 상단에서 한국어 / English 전환 (선택 저장)
- 📱 **모바일 완전 지원** — 반응형 레이아웃 + 터치 컨트롤(가상 D-패드·버튼·스와이프·드래그)
- 💾 게임 대부분이 `localStorage`에 최고기록 저장

## 📱 모바일 대응 방식

- `games/_arcade.css` — 반응형 레이아웃, 캔버스 자동 축소, 안전영역(노치) 패딩, 터치 컨트롤 스타일
- `games/_controls.js` — 화면 터치 버튼이 **실제 키보드 이벤트를 발생**시켜 기존 게임 로직을 그대로 재사용. 터치 기기에서만 표시되고, 마우스 데스크톱에서는 자동 숨김
- 캔버스 기반 게임은 추가로 스와이프·드래그·탭 등 직접 터치 입력 지원

## 🚀 GitHub Pages 배포

1. 이 폴더 전체를 GitHub 저장소에 푸시합니다.
2. **Settings → Pages → Branch**를 `main` / `(root)` 으로 설정.
3. `https://<사용자명>.github.io/<저장소명>/` 에서 메인 페이지가 열립니다.

> 로컬에서는 `index.html`을 더블클릭하면 바로 실행됩니다.

## 📁 구조

```
index.html              # 메인 메뉴 (게임 카드 27개, 언어 토글, Made by DH)
games/
 ├─ _arcade.css         # 공통 반응형/터치 스타일
 ├─ _controls.js        # 공통 터치 컨트롤러 (터치 → 키 이벤트)
 ├─ shooter · racing · mario · fortress · minesweeper · whackamole · tetris
 ├─ snake · breakout · 2048 · flappy · memory · invaders · connect4
 ├─ simon · asteroids · bubble · dino · pacman · pong · slidingpuzzle
 └─ sudoku · reaction · math · match3 · stack · maze   (.html)
```

각 게임은 단일 HTML 파일에 CSS/JS가 인라인되어 있고, 공통 모바일 자산(`_arcade.css` / `_controls.js`)만 공유합니다.
